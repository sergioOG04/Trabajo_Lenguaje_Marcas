Apache libreoffice
Apache OpenOffice es una suite de oficina de código abierto líder para el procesamiento
de palabras, hojas de cálculo, presentaciones, gráficos, bases de datos y más.
Se encuentra disponible en varios idiomas y funciona en todos los sistemas comunes.		 LibreOffice incorpora varias aplicaciones que lo convierten en la más potente
suite de oficina Libre y de Código Abierto del mercado: Writer, el procesador de textos,
Calc, la hoja de cálculos, Impress, el editor de presentaciones, Draw,
nuestra aplicación de dibujo y diagramas de flujo, Base, nuestra base de datos
